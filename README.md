# Decision Tree Regression

## Overview

This project implements Decision Tree Regression using Scikit-learn. The program performs preprocessing, trains the model, evaluates its accuracy, visualizes the regression curve, and predicts values using user input.

## Features

- Load CSV dataset
- Handle missing values
- Encode categorical variables
- Select target column
- Split dataset
- Train Decision Tree Regressor
- Predict test data
- Calculate R² Score
- Plot regression graph
- User input prediction

## Technologies

- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

## Files

- Decision Tree.ipynb
- Position_Salaries.csv
- Decision Tree Regression.docx

## How to Run

1. Install dependencies.
2. Open notebook.
3. Run each cell.
4. Enter required information.
5. View prediction and graph.

## Output

- Decision Tree Model
- Prediction
- R² Score
- Regression Plot
